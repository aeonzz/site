## Overview

Grydal is an image gallery focused on getting out of the way. I built it around a simple browsing flow: open the page, scan the photos, and keep moving without fighting the interface.

---

## Highlights

The layout keeps the spacing steady and the chrome quiet, so the photos stay in front. Nothing on the page is trying to compete with the image grid.

I tuned the grid around the way people actually browse galleries: quick passes first, closer looks second. The result is dense enough to scan, but not so packed that the page starts to feel noisy.

---

## Tech Stack

The stack keeps the gallery lightweight: Next.js and React for the interface, the Pexels API for image data, and Tailwind CSS for the dense responsive layout.
